# Atlas Anatomi Skeletal

Website pembelajaran interaktif anatomi skeletal yang berisi materi, model 3D, dan kuis interaktif.

## Fitur
- Materi anatomi skeletal
- Navigasi regio anatomi
- Kuis interaktif
- Watermark identitas mahasiswa
- Logout dan penyimpanan hasil kuis

## Cara Menjalankan
Buka file [Untitled-1.html](Untitled-1.html) di browser.
